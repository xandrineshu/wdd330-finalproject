# 🌸 PokéDeck: A Pastel Portfolio Project

Welcome to **PokéDeck**! A sleek, responsive, and high-performance Pokémon discovery tool built
with a cute pastel aesthetic. This project was developed as part of a WDD (Web Design and Development) 
portfolio to showcase API integration, recursive logic, and modern UI/UX design.



## ✨ Key Features
* **🔍 Smart Search:** Real-time search suggestions as you type to prevent typos.
* **💖 Personal Deck:** Save your favorite Pokémon to a "Favorites" gallery powered by LocalStorage.
* **🧬 Evolution Tracking:** A recursive algorithm that maps out complex evolution chains.
* **✨ Interactive Polish:** Custom JS sparkle particle systems and smooth CSS animations.
* **📱 Fully Responsive:** Designed with CSS Grid and Flexbox to look great on any device.


## 🛠️ Built With
* **HTML5 & CSS3** (Custom Root Variables & Neo-Brutalism style)
* **Vanilla JavaScript** (ES6+)
* **PokéAPI** (Data Source)
* **WebP/SVG** (High-performance asset optimization)
